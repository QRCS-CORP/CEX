Initialize the default structure.

Default settings are sequential mode.

Parameters
OutputSizeDigest output byte length; set to 32 for Blake2-256, or 64 for Blake2-512
TreeDepthMaximal depth (1 byte): an integer in [1, 255] (set to 255 if unlimited, and to 1 only in sequential mode)
FanoutThe number of state leaf-nodes used by parallel processing (limit of one state per processor core is recommended)
LeafSizeThe outer leaf length in bytes; set to 0 for unlimited
InnerLengthInner hash byte length (1 byte): an integer in [0, 64] for BLAKE2b, and in [0, 32] for BLAKE2s(set to 0 in sequential mode)
